# DzawaniKost

Website kost modern dan responsif dibangun dengan Next.js 16, TypeScript, dan Tailwind CSS v4.

## Fitur

- **Beranda** - Hero section, fasilitas umum, kamar pilihan, harga, testimoni, dan lokasi
- **Daftar Kamar** - Semua kamar dengan filter tipe dan status ketersediaan
- **Detail Kamar** - Galeri foto, fasilitas lengkap, dan tombol kontak WhatsApp
- **Tentang Kami** - Profil, nilai-nilai, peraturan kost, dan tim pengelola
- **Kontak** - Form pesan, info kontak, dan jam operasional

## Teknologi

- [Next.js 16](https://nextjs.org/) - React Framework
- [TypeScript](https://www.typescriptlang.org/) - Type Safety
- [Tailwind CSS v4](https://tailwindcss.com/) - Styling
- [Lucide React](https://lucide.dev/) - Icons

## Menjalankan Secara Lokal

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build

# Jalankan production server
npm start
```

Buka [http://localhost:3000](http://localhost:3000) di browser Anda.

## Deploy di Vercel

Project ini sudah dikonfigurasi untuk deploy di Vercel. Cukup hubungkan repository GitHub ke Vercel dan deploy otomatis akan berjalan setiap kali ada push ke branch main.
